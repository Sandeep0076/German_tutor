## Exam Day Tips

### Listening (H√∂ren)
‚úÖ Read all questions BEFORE audio starts
‚úÖ Mark W-words (Wann? Wo? Was? Wer? Wie?)
‚úÖ Don't panic if you miss something - keep moving forward
‚úÖ Use pictures as clues (Teil 1)
‚úÖ Write down numbers/times immediately
‚úÖ Teil 2 is played only ONCE - focus extra hard!

### Reading (Lesen)
‚úÖ Skim the text first, then read questions
‚úÖ Look for keywords in both question and text
‚úÖ Don't need to understand every word
‚úÖ Time management: ~8 minutes per Teil
‚úÖ Eliminate obviously wrong answers first

### Writing (Schreiben)
‚úÖ Teil 1: Print clearly, check spelling
‚úÖ Teil 2: Cover ALL 3 points mentioned in task
‚úÖ Keep it simple - don't try complex sentences
‚úÖ Use familiar vocabulary
‚úÖ Leave 2-3 minutes to check for errors
‚úÖ Standard structure: Greeting ‚Üí 3 points ‚Üí Closing ‚Üí Signature

### Speaking (Sprechen)
‚úÖ Speak clearly and slowly
‚úÖ If you don't understand, ask: "Entschuldigung, k√∂nnen Sie das wiederholen?"
‚úÖ Teil 1: Spell your name letter by letter
‚úÖ Teil 2: Both ask AND answer questions
‚úÖ Teil 3: Make requests politely: "Kann ich bitte..."
‚úÖ Don't worry about small grammar mistakes - communication matters more!
