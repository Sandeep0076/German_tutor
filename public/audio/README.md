# Audio Files for German Tutor A1 Exam Focus

This directory contains audio files for the listening practice exercises.

## Required Audio Files

### Day 01 - Short Dialogues
The following audio files are referenced in the Day 1 listening exercises:

1. **dialogue1.mp3** - Supermarket conversation
   - Used for questions 1-2
   - Content: Conversation about buying bread and cheese at a supermarket
   - German dialogue example:
     ```
     Person A: Guten Tag! Ich brauche Brot und Käse.
     Person B: Sehr gerne. Das Brot ist hier, und der Käse ist dort.
     ```

2. **dialogue2.mp3** - Doctor's appointment
   - Used for questions 3-4
   - Content: Making a doctor's appointment for Wednesday at 10:30
   - German dialogue example:
     ```
     Person A: Guten Tag, ich brauche einen Termin beim Arzt.
     Person B: Am Mittwoch haben wir noch Zeit. Um halb elf?
     Person A: Ja, das passt gut. Danke!
     ```

3. **dialogue3.mp3** - Meeting someone new
   - Used for questions 5-6
   - Content: Introduction, person from Spain living in Berlin for 2 months
   - German dialogue example:
     ```
     Person A: Hallo! Woher kommst du?
     Person B: Ich komme aus Spanien. Ich wohne seit zwei Monaten hier in Berlin.
     ```

## How to Add Audio Files

### Option 1: Record Your Own Audio
1. Use a text-to-speech service or record native speakers
2. Save files as MP3 format
3. Place them in the appropriate day folder (e.g., `public/audio/day01/`)

### Option 2: Use Text-to-Speech Services
- Google Cloud Text-to-Speech (German voice)
- Amazon Polly (German voice)
- Microsoft Azure Speech Service

### Option 3: Temporary Placeholder
Until real audio files are available, you can:
1. Create silent MP3 files as placeholders
2. Use the application's fallback text display
3. Generate audio using browser's Web Speech API

## File Naming Convention
- Format: `dialogue[number].mp3`
- Example: `dialogue1.mp3`, `dialogue2.mp3`, etc.
- Location: `public/audio/day[number]/`
